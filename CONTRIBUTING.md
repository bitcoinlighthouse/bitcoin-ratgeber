# Mitmachen

Danke für dein Interesse, zum Bitcoin-Ratgeber beizutragen!

## Was willkommen ist

- **Fehler melden** – Tippfehler, sachliche Fehler, veraltete Links → Issue öffnen
- **Verbesserungsvorschläge** – Formulierungen, Ergänzungen, fehlende Themen → Issue oder Pull Request
- **Übersetzungen** – Andere Sprachen sind ausdrücklich erwünscht. Bitte vorher kurz per Issue ankündigen, damit keine doppelte Arbeit entsteht.

## Wie du einen Pull Request erstellst

1. Repository forken
2. Änderung in deinem Fork vornehmen
3. Pull Request mit kurzer Beschreibung öffnen

## Stil-Hinweise

- Zielgruppe sind Bitcoin-Einsteiger – bitte einfache, klare Sprache
- Keine Altcoin-Empfehlungen
- Links zu externen Diensten bitte kurz begründen
